pmt:
	Ferramenta para busca de padrões num arquivo ou em um conjunto de arquivos que utiliza o algoritmo de sellers para busca aproximada e kmp para busca exata. Para mais informações de como usa-la: rodar o executável com a opção -h.

	Daniel Maida   - dmf2@cin.ufpe.br
	
Intruções para compilação:
	~ pmt/src$ make
